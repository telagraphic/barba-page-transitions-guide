Rules

You can set the rule for a transition to apply to certain pages with a from and to block in your transition. In this example, going from the /index to the /guide page uses a transition. Going from /index to /resources uses a different one. Going to /guide from any other page uses it's own transition.

Look at the console statements in the console tab to see the order of hook events. leave runs when leaving from /index and enter will run when going to /guide. It's easy to get confused when a hook will run when adding rules.

/guide to /resources does nothing! Try something out for it.
