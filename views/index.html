Synchronous

The below code is synchronous, it runs but Barba does not know it is complete. The end result is that no beautiful transition code is run and pages just load the standard way old fashion way.

If your new to Barba and async code, you might try writing your hooks this way. Now you know what it doesn't work.

Click on the links to see how lame it is. 

            
              barba.init({
                transitions: [
                  {
                    sync: true,
                    name: 'fade',
                    enter({current, next, trigger}) {
                      const timeline = gsap.timeline();
                      timeline
                        .to(current.container, {opacity: 1, ease: 'circ'});
                    },
                    leave({current, next, trigger}) {
                      const timeline = gsap.timeline();
                      timeline
                        .to(current.container, {opacity: 0, ease: 'circ'});
                    }
                  }
                ],
                debug: true
              });
